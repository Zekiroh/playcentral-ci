# PlayCentral

Tournament registration system built with CodeIgniter 4 as an academic project.

## Tech Stack
- PHP
- CodeIgniter 4
- MySQL

## Setup (Local)
1. Import `playcentral_db.sql` into MySQL.
2. Copy `.env.example` to `.env` (or configure your `.env` locally) and set DB credentials.
3. Run the app locally (point your server to the `public/` folder).
